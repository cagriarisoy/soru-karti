Bu klasör, gerçek backend kullanıldığında kullanıcı JSON dosyalarının tutulabileceği örnek klasördür.

GitHub Pages statik yayın yaptığı için tarayıcıdan bu klasöre otomatik dosya yazamaz. Bu sürümde kullanıcı verileri tarayıcıda kullanıcı adına özel localStorage anahtarıyla tutulur ve kullanıcı isterse kendi JSON dosyasını dışa aktarır.
